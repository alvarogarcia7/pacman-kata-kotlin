# Pacman Kata (in Kotlin)

## Responsibilities

  * Grid
  	* size
  	* paths[]
  	* bonus[]
  		* x,y
  * Pacman
  	* x,y, direction
  	* state (dead/alive/eating)
  * Monsters
  	* x,y, direction
  	* state (dead/alive/being eaten)
  	* movement strategy
  * Statistics
  	* level
  	* score
  	* max_score
